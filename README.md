# Dotfiles

## Installation

### MacOS
```
brew install coreutils
git clone https://github.com/rwejlgaard/dotfiles ~/dotfiles
cd ~/dotfiles && ./dotdrop.sh install --profile base
```

### Linux
```
git clone https://github.com/rwejlgaard/dotfiles ~/dotfiles
cd ~/dotfiles && ./dotdrop.sh install --profile base
```
